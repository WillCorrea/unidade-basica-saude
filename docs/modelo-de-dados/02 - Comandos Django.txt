Criar:
	um Management Command Django: seed_rbac
	Ele cria:
		Groups: ADMIN, RT, ESTOQUE, ATENDENTE, AUDITOR
		Permissions (do Django) e liga nos Groups

E o “seed de admin” vira também command
	cria tenant + ubs + stock_location
	cria superuser/admin
	vincula admin à UBS via user_ubs_groups

✅ Resultado: tudo padronizado e executável com:
python manage.py seed_rbac
python manage.py seed_initial


Estrutura do projeto Django (arquivos/pastas)

Crie o projeto e apps assim:
	django-admin startproject config .
	python manage.py startapp core
	python manage.py startapp accounts
	python manage.py startapp catalog
	python manage.py startapp stock
	python manage.py startapp operations
	python manage.py startapp audit


Responsabilidades:

	core: tenants, ubs, stock_locations
	accounts: user_ubs_groups + helpers de permissão por UBS
	catalog: medicines, batches
	stock: stock_balances, stock_movements, stock_movement_items, movement_reversals
	operations: invoices, dispensations, prescriptions, inventories
	audit: audit_logs
